## ðŸ“• Level 4: Advanced (Days 13-18)

### Exercise 4.1: Responsive Navigation

Create a navbar that:

- Shows hamburger menu on mobile
- Shows full menu on desktop
- Has dropdown hover effect
- Sticky at the top when scrolling

**Use:** `md:hidden`, `md:flex`, `sticky`, `top-0`

### Exercise 4.2: Image Card with Overlay

Create a card with:

- Background image
- Dark overlay (50% opacity)
- Text content on top
- Hover effect: overlay becomes darker
- Button at bottom

**Use:** `relative`, `absolute`, `bg-opacity-50`, `group`, `group-hover`

### Exercise 4.3: Modal Component

Build a modal that:

- Centers on screen
- Has backdrop (dark, semi-transparent)
- Contains a form
- Has close button
- Appears with fade-in animation

**Use:** `fixed`, `inset-0`, `backdrop-blur`, `transition`, `opacity`
